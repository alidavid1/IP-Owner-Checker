
# IP Address Processor

This application allows you to process IP addresses provided as plain text and generate an Excel report.

## Features
- Accepts IP addresses as text input.
- Outputs results in an Excel file.
- Two usage options:
  - **Python Script**: Run the script directly if you have Python installed.
  - **Executable File**: Download the executable version with a simple GUI for ease of use.

## How It Works
1. Provide IP addresses as plain text.
2. The application processes the input.
3. An Excel file is generated as output.
